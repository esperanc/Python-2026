# O que é programação competitiva?

É um tipo de competição em que os participantes resolvem problemas de programação.

No Brasil, a principal competição é a [Maratona SBC de Programaçãp](https://maratona.sbc.org.br/sobre/index.html)

Uma das principais plataformas para resolver problemas de programação competitiva é o [Beecrowd](https://judge.beecrowd.com)

---
# Usando a plataforma Beecrowd

Para resolver problemas no Beecrowd, é necessário criar uma conta no site https://judge.beecrowd.com.

O site vai pedir para você entrar com seus dados (bem chato, eu sei).

Uma vez logado, os principais botões são:

- "Problemas": para selecionar problemas para resolver
- "Submissões": para ver o resultado das suas submissões

# Preparando uma submissão

Use o Jupyterlite, VS-Code ou seu editor favorito para criar o código para sua resposta.

Seu código também deve conter comandos `input` para receber os dados de entrada, e `print` para imprimir os dados de saída.

Preste bastante atenção ao _formato_ dos dados de entrada e de saída.

Teste com os dados de entrada fornecidos no problema. 

Para alguns problemas, a _eficiência_ do código também é avaliada.

Quando estiver satisfeito com o código, copie-o e cole no Beecrowd e clique em "Enviar".
